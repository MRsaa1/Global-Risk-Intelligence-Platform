# 🌐 Правильные адреса для доступа к системе

## ⚠️ Важно: порт 9001 занят MinIO

**Порт 9001 используется MinIO** (объектное хранилище, не наш сервис).  
**НЕ используйте порт 9001** для доступа к нашей системе!

---

## ✅ Правильные адреса

### 📊 Control Tower UI (React приложение)

#### Главная страница
```
http://localhost:9000
```
- Требует аутентификации
- Username: `demo` (или любой)
- Password: `demo` (или любой)

#### Demo Page (рекомендуется для первого просмотра)
```
http://localhost:9000/demo
```
- ✅ Не требует аутентификации
- ✅ Показывает все функции
- ✅ Графики и визуализации
- ✅ Демо-данные

### 🔌 API Gateway (Backend API)

#### Health Check
```
http://localhost:9002/health
```
Ответ:
```json
{
  "status": "ok",
  "timestamp": "2025-11-17T..."
}
```

#### Demo Data
```
http://localhost:9002/api/v1/demo/data
```
Возвращает:
- Dashboard метрики
- Scenarios
- Calculations
- Portfolios

#### Demo Metrics
```
http://localhost:9002/api/v1/demo/metrics
```
Возвращает метрики за 30 дней для графиков

#### Demo Login
```
POST http://localhost:9002/api/v1/demo/login
Body: { "username": "demo", "password": "demo" }
```

---

## ❌ НЕ используйте эти адреса

### Порт 9001 (MinIO)
```
http://localhost:9001/login  ❌
http://localhost:9001/demo   ❌
```
Это MinIO (объектное хранилище), не наш сервис!

### Другие зарезервированные порты
- `3000` - занят
- `5174` - занят
- `8080` - занят
- `3001` - занят
- `8501` - занят
- `5001` - занят
- `9001` - занят MinIO

---

## 🎯 Быстрый старт

1. **Откройте Demo Page** (без аутентификации):
   ```
   http://localhost:9000/demo
   ```

2. **Или войдите в систему**:
   ```
   http://localhost:9000
   Username: demo
   Password: demo
   ```
