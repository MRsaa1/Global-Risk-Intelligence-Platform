## 🎯 Рекомендуемые следующие шаги

### Вариант 1: Быстрый старт (для демо/тестирования)
**Цель**: Получить работающую систему за 1-2 дня

1. **База данных - простые модели**
   - SQLAlchemy модели для сценариев, расчетов, портфелей
   - Простые CRUD операции
   - In-memory или SQLite для быстрого старта

2. **Интеграция API с данными**
   - Подключить реальные запросы к БД
   - Заменить заглушки на реальные вызовы

3. **Тестирование end-to-end**
   - Проверить полный flow: создание сценария → запуск расчета → просмотр результатов

### Вариант 2: Production-ready (для реального использования)
**Цель**: Готовая к продакшену система за 1-2 недели

1. **База данных (3-4 дня)**
   - PostgreSQL с SQLAlchemy
   - Alembic миграции
   - Индексы и оптимизация
   - Seed данные

2. **Аутентификация (2-3 дня)**
   - Полная JWT реализация
   - OIDC/SAML интеграция
   - RBAC (роли и права)
   - Управление пользователями

3. **Мониторинг (2-3 дня)**
   - OpenTelemetry
   - Prometheus метрики
   - Структурированное логирование
   - Health checks

4. **Интеграция сервисов (2-3 дня)**
   - API Gateway → reg-calculator
   - API Gateway → data adapters
   - Очереди для асинхронных расчетов
