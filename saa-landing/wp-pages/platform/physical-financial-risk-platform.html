About

Physical-Financial Risk Platform bridges the gap between physical reality and financial models. Every change in the physical world maps to the financial model — and back — in real time, automatically, and verifiably. The platform combines 3D Digital Twins with climate simulations, financial models, and a dependency network (Knowledge Graph). Product focus: physical asset risk, credit/insurance for complex assets, REIT/portfolios, fraud detection, project finance, immersive analytics. Supports climate stress testing (TCFD/NGFS), BCP/Recovery Plans automation, and NVIDIA AI (Earth-2, PhysicsNeMo, LLM).

Command Center

The main interface is Command Center. After starting the stack locally, open: http://127.0.0.1:5180/command

  • Unified screen: 3D globe (Cesium), risk zones, city and scenario selection
  • Stress tests: seismic, flood, hurricane, climate, financial, geopolitical, cyber, pandemic; Critical/High/Medium/Low zones
  • Cascade Analysis: dependency graph, risk propagation (BFS/GNN), «Open in Cascade» from reports
  • Digital Twins: 3D BIM/IFC, stress tests by scenario, export to Cascade
  • Panels: Zone Metrics, Stress Metrics, Timeline Predictions, Cascade Flow, Command Mode
  • Historical events: comparison with past crises (1929, 2008, 2022, etc.)
  • Agents and monitoring: SENTINEL, ANALYST, ADVISOR, REPORTER; Agent Monitoring and System Overseer widgets
  • Reports: PDF, HTML, LLM summaries, action plans

Five layers of reality

  • Layer 0 — Verified Truth: Cryptographic proofs of physical state, immutable audit trail
  • Layer 1 — Living Digital Twins: 3D BIM (IFC), temporal history, IoT, geometry, condition, exposures, financials
  • Layer 2 — Network Intelligence: Knowledge Graph (Neo4j), dependency discovery, cascade risk, infrastructure links
  • Layer 3 — Simulation Engine: Physics (flood, structural, thermal, fire), Climate (CMIP6, acute/chronic), Economics (PD, LGD, DCF), Cascade (Monte Carlo)
  • Layer 4 — Autonomous Agents: SENTINEL, ANALYST, ADVISOR, REPORTER, SYSTEM OVERSEER (health, circuit breakers)
  • Layer 5 — Protocol (PARS): Physical Asset Risk Schema, interoperability

Technology stack

Frontend

React 18, TypeScript, Vite, Framer Motion, CesiumJS (3D globe), Deck.gl, Three.js/R3F, TailwindCSS. Command Center, Dashboard, Analytics, BIM viewer.

Backend

Python FastAPI, Celery + Redis, GraphQL, WebSockets. REST + GraphQL. PostgreSQL + PostGIS, TimescaleDB, Neo4j, Redis, MinIO.

3D / BIM / AI

IFC.js, IfcOpenShell, Open3D, CesiumJS. PyTorch, PyG (GNN), LangChain, NVIDIA NIM. CMIP6, FEMA, NOAA, Copernicus, NVIDIA Earth-2.

Resilience & infra

Circuit Breakers, Retry with Backoff, Fallback. System Overseer, health checks. Docker, Kubernetes, auto-restart scripts.

Quick start

Prerequisites: Python 3.11+, Node.js 18+, PostgreSQL, Redis (or Docker). Option 1: docker compose up -d postgres redis neo4j minio; then cd apps/api && uvicorn src.main:app --reload --port 9002; cd apps/web && npm run dev. Option 2: ./start-all-services.sh. Then open Command Center at http://127.0.0.1:5180/command. See How it works and project README for details.
