About

ARIN Platform is a full-featured risk management and analytics aggregation system from SAA Alliance. It unifies 6 specialized AI risk agents, graph analysis, LLM reasoning, ML models, a single verdict from multiple data sources, authentication, compliance, and rich visualizations. Each agent has its own model (DeepSeek R1, Llama, Mistral via NVIDIA API) with GPT-4 fallback; graph analysis covers dependencies, cascade effects, and GNN for systemic risk (PyTorch Geometric).

Capabilities

Risk agents and AI

  • 6 specialized risk agents — Credit, Market, Operational, Liquidity, Regulatory, Systemic; each with its own model and fallback
  • Graph analysis — Dependencies, cascade effects, GNN for systemic risk (PyTorch Geometric)
  • LLM and ML — Response caching, retry, multi-provider; XGBoost/LightGBM for credit risk, GNN for network analysis

Unified integration and export

  • Unified Integration — Single data feed from Risk Analyzer, Investment Dashboard, News/Crypto Analytics, AI Trader; final verdict (BUY/SELL/HOLD/AVOID) and risk score across sources
  • Export API — Accept exports from external SAA Alliance projects, persistent storage, entity binding (portfolio, symbol)

Auth, compliance, and infrastructure

  • Authentication — Registration, login, JWT, OAuth; users and roles in PostgreSQL (Alembic migrations)
  • Compliance — GDPR (data access, export, deletion), audit log, retention policies, backup and recovery
  • Visualizations — Graphs (2D/3D force-graph), ECharts, D3.js, Plotly 3D, Three.js, Recharts; risk dashboards
  • Integrations — NVIDIA Build panel, Superset, portfolio sync, alerts, config storage (NVIDIA, Superset)
  • Infrastructure — Cache (Redis), queues (Celery), monitoring, health checks, Docker/Kubernetes

Architecture

Top layer: Auth (JWT, OAuth), Compliance (GDPR, Audit, Backup), Config API. Orchestrator distributes tasks across agents. Six agents: Credit Risk (Llama 405B), Market Risk (DeepSeek R1), Operational (Llama 70B), Liquidity (DeepSeek), Regulatory (Llama 405B), Systemic (DeepSeek + GNN). Graph Builder (dependencies, clusters, paths) + GNN. AI Engine: LLM Manager (NVIDIA API, OpenAI), ML models, guardrails. Unified: Aggregator, Verdict Generator, Export Storage, API keys. Data: PostgreSQL (users, roles), Redis (cache), Neo4j (graph).

Technology stack

Backend

Python 3.11+, FastAPI, Uvicorn, Pydantic 2; SQLAlchemy 2.x, asyncpg, Alembic; PostgreSQL. Redis, Celery. Neo4j, NetworkX, PyVis. JWT (python-jose), Passlib (bcrypt), OAuth-ready.

AI / ML

LLM: OpenAI-compatible API (NVIDIA NIM: DeepSeek R1, Llama, Mistral), GPT-4 fallback. ML: scikit-learn, XGBoost, LightGBM, PyTorch, PyTorch Geometric (GNN), transformers. Data: pandas, numpy, polars, scipy.

Frontend

Next.js 14, React 18, TypeScript. Tailwind CSS. ECharts, D3.js, Plotly, Three.js, Recharts, react-force-graph-2d/3d. Zustand, TanStack React Query, Axios.

Infrastructure

Docker, Docker Compose, Kubernetes. CORS, optional nginx, SSL.

Quick start

Requirements: Python 3.11+ (3.12 recommended), PostgreSQL 15+, Node.js 18+. Clone arin-platform, run ./DO_EVERYTHING.sh (DB, .env, venv, migrations). Backend: uvicorn backend.main:app --reload → API http://localhost:8000, docs http://localhost:8000/docs. Frontend: cd frontend && npm install && npm run dev → http://localhost:3000. Docker: docker-compose up -d.

Main API

Base: /api/v1

  • Health — GET /health, GET /api/v1/performance/health
  • Agents — GET /api/v1/agents, GET /api/v1/agents/{id}
  • Analysis — POST /api/v1/risks/analyze (market_analysis, credit_analysis, liquidity_analysis, …)
  • Graph — GET /api/v1/graph, GET /api/v1/graph/visualization
  • Auth — POST /api/v1/auth/register, POST /api/v1/auth/login, GET /api/v1/auth/me
  • Unified — POST /api/v1/unified/export, GET /api/v1/unified/exports/{entity_id}, GET /api/v1/unified/verdict/{entity_id}
  • Alerts — GET /api/v1/alerts
  • LLM — POST /api/v1/llm/generate, POST /api/v1/llm/extract
