About

Crypto Analytics Portal is a full-featured institutional digital asset analytics platform. It combines multi-metrics, AI analytics (NVIDIA DeepSeek R1, Llama), a compliance map, a trading dashboard with TradingView, ARIN Platform integration, and multi-language report generation. The service grew from a metrics dashboard into a full system: real-time data, WebSocket, AI summaries and sentiment, report delivery to ARIN, economic calendar, impact graph, and an extended trading block.

Capabilities

Multi-metrics and rating

  • Fundamentals, On-Chain, Market & Sentiment, Security, Regulatory, ESG
  • Weighted model v1.4, investment rating (AA)

AI analytics

  • Executive Summary in reports — NVIDIA DeepSeek R1 (reasoning, streaming)
  • Sentiment by ticker — Llama 3.1 8B (NVIDIA API)
  • Trading Insight — AI insights in the trading dashboard

Trading dashboard

  • TradingView widgets, real-time prices (WebSocket)
  • Exchange flow, whale activity, orderbook, momentum, DEX volume
  • Fear & Greed, economic calendar, AI insight per asset

Reports and ARIN

  • Institutional HTML/PDF reports, EN/RU
  • Send to ARIN — send analysis to ARIN Platform
  • Report window and offline HTML download

Visualization and backend

  • Interactive Compliance Map by region
  • Impact Graph Network (vis-network) — metric influence graph
  • Price charts and price block. CoinGecko (live + cache), multi-API layer, ARIN client. saa-metrics module (hexagonal: domain, adapters, usecase). Rate limiting, CORS, CSP, WebSocket hub.

Technology stack

Frontend

React 18, TypeScript. TailwindCSS, Framer Motion. React Router, Lucide React. Three.js, @react-three/fiber, @react-three/drei (Earth 3D). vis-network (Impact Graph). MetricCard, ComplianceMap, TradingViewSection, PriceChartWidget, AISummary, ImpactGraphNetwork, ReportButton, ReportWindow.

Backend

Go 1.24, Gin. CoinGecko API, multi-API, ARIN client. NVIDIA API: DeepSeek R1, Llama 3.1 8B (sentiment), Dracarys 70B (fallback). Gorilla WebSocket, rate limiting. SPA: static from build/, NoRoute → index.html.

saa-metrics (hexagonal)

domain (asset, metrics, ports), adapters (fundamentals, onchain, market, security, regulatory, esg), usecase (assessment, scoring), delivery/http.

Production

Contabo. Live: crypto.saa-alliance.com. Nginx: proxy /api and /ws to 8083, serve SPA from build. SSL via Certbot.

Institutional rating (v1.4)

Fundamentals 30% (Market Cap, Dev Activity, Governance); On-Chain 20% (Active Addresses, Volume, TVL); Market & Sentiment 20% (Volatility, BTC Correlation, Sentiment); Security 15% (Audits, Bug Bounty, Incidents); Regulatory 10% (US, EU, Sanctions); ESG 5% (ESG Score, DAO, Transparency).

API (main)

Base: /api

  • GET /api/live/:symbol — Real-time metrics
  • GET /api/asset/:symbol/metrics — Dashboard metrics (aggregate)
  • GET /api/analyze/:symbol — Asset analysis
  • GET /api/tickers/top-100 — Top 100 tickers
  • GET /api/crypto/:symbol/report?lang=en|ru — HTML report (with AI summary)
  • GET /api/download/report/:symbol — Report download
  • POST /api/send-to-arin/:symbol — Send to ARIN Platform
  • GET /api/sentiment/:symbol — AI sentiment (Llama 8B)
  • GET /api/trading/exchange-flow/:symbol, whale-activity, orderbook, momentum, dex-volume, fear-greed, insight/:symbol, economic-events
  • GET /api/ws — WebSocket (real-time prices). POST /api/cache/clear — Clear report cache

Quick start

Prerequisites: Node.js 18+, Go 1.24+. Frontend: npm install, npm start (dev), npm run build (production). Backend: go mod download, go build -o saa-backend main.go, ./saa-backend (port 8083). .env: COINGECKO_API_KEY, NVIDIA_API_KEY_CRYPTO, PORT, GIN_MODE, REACT_APP_API_URL.

Security

CORS whitelist in production; CSP (different policies for reports with TradingView and for SPA); rate limiting; X-Content-Type-Options, X-Frame-Options, Referrer-Policy; keys only via environment variables.
