About

AI-Powered Prediction Market Protocol is the first AI-native prediction market: AI agents automatically create relevant markets from trends; agents analyze real-time data (social, news, on-chain); users get insights via analytical agents; agents can trade with each other for deep liquidity; prediction market prices become a monetizable data asset for institutional clients.

Project goal

Demonstrate ability to design and implement complex DeFi protocols: LMSR (Logarithmic Market Scoring Rule) for prediction markets, multi-contract architecture with governance, oracle integration and security best practices, full-stack Web3 application.

Overview

  • AI agents automatically create relevant markets from trends
  • Agents analyze data in real time (social, news, on-chain metrics)
  • Users get insights through analytical agents
  • Agents can trade with each other for deep liquidity
  • Prediction market prices become monetizable data for institutional clients

Architecture

Protocol layers:

  • Smart Contracts (Solidity) — core on EVM
  • Frontend (Next.js + React + TypeScript) — web UI
  • Backend (Node.js + TypeScript) — API and workers
  • Indexing (The Graph + custom indexer) — event indexing
  • AI Layer — AI agents and data processing

Project structure

ai-prediction-market/: contracts/ (Foundry — core, oracle, ai, staking, governance, treasury), frontend/ (Next.js), backend/ (Node.js API), indexer/ (The Graph subgraph + custom indexer), docs/.

Quick start

Requirements: Node.js 20+, Foundry (contracts), PostgreSQL 15+, Redis 7+. Contracts: cd contracts && forge build && forge test; deploy to local: forge script script/Deploy.s.sol --rpc-url http://127.0.0.1:8545 --broadcast. Frontend: cd frontend && npm install && npm run dev. Backend: cd backend && npm install && npm run dev, npm run migrate, npm run workers.

Tokenomics

Token: PRED (ERC-20 with ERC-2612 permit and ERC-20Votes). Total supply: 1,000,000,000 PRED. Distribution: Team 20% (4y vesting), Investors 15% (4y vesting), Community 40%, Treasury 15%, Liquidity 10%.

Networks

Polygon — main network for prediction markets (USDC). Base — AI agents. LayerZero — cross-chain bridge.
