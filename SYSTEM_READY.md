## 📱 Что можно протестировать

1. **Откройте Demo Page**
   - http://localhost:3000/demo
   - Увидите все функции сразу

2. **Войдите в систему**
   - http://localhost:3000
   - Username: `demo`, Password: `demo`

3. **Исследуйте Dashboard**
   - Метрики в карточках
   - Графики трендов
   - Таблицы данных

4. **Создайте сценарий**
   - Scenarios → Create Scenario

5. **Запустите расчет**
   - Calculations → Start Calculation
   - Наблюдайте real-time обновления

6. **Просмотрите результаты**
   - Нажмите на иконку просмотра
   - Изучите графики и таблицы

7. **Экспортируйте отчет**
   - На странице результатов
   - Export PDF или Excel
