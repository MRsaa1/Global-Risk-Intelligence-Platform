## 📊 Что можно увидеть

### 1. Dashboard (Главная)
- **4 градиентные карточки**:
  - 📋 Активные сценарии (фиолетовый)
  - ⚡ Расчеты в процессе (розовый)
  - ✅ Завершенные расчеты (синий)
  - 💼 Портфели (зеленый)

- **Интерактивные графики**:
  - Line Chart: тренды Capital Ratio и LCR
  - Bar Chart: сравнение compliance

- **Таблицы**:
  - Активные сценарии
  - Последние расчеты
  - Портфели

### 2. Demo Page
- Все функции на одной странице
- Графики трендов (30 дней)
- Демо-данные для всех компонентов

### 3. Scenarios
- Создание сценариев
- Список всех сценариев
- Редактирование и удаление

### 4. Calculations
- Запуск расчетов
- Real-time обновления статуса
- Просмотр результатов
- Детальная страница с графиками

### 5. Calculation Detail
- **4 вкладки**:
  - Overview: графики и метрики
  - Basel IV: результаты Basel IV
  - Liquidity: метрики ликвидности
  - Details: полные детали

- **Графики**:
  - Pie Chart: структура капитала
  - Bar Chart: метрики соответствия

- **Экспорт**:
  - Export PDF
  - Export Excel

### 6. Portfolios
- Список портфелей
- Агрегированные метрики
- Форматирование валют

## 🎓 Что протестировать

1. **Откройте Demo Page**
   - http://localhost:3000/demo
   - Увидите все функции сразу

2. **Войдите в систему**
   - http://localhost:3000
   - Username: `demo`, Password: `demo`

3. **Создайте сценарий**
   - Scenarios → Create Scenario

4. **Запустите расчет**
   - Calculations → Start Calculation
   - Наблюдайте real-time обновления

5. **Просмотрите результаты**
   - Нажмите на иконку просмотра
   - Изучите графики и таблицы

6. **Экспортируйте отчет**
   - На странице результатов
   - Export PDF или Excel
