# Как применить правки глобуса (города по чекбоксам, синие круги, круги над рельефом)

Код уже изменён в репозитории. Если в браузере по-прежнему старый вид — загружается **старый бандл**. Сделайте по шагам:

## 1. Остановите dev-сервер фронта

В терминале, где запущен `npm run dev` (apps/web), нажмите **Ctrl+C**.

## 2. Очистите кэш и перезапустите

Из **корня репозитория** (global-risk-platform):

```bash
cd /Users/artur220513timur110415gmail.com/global-risk-platform
./apps/web/rebuild-globe.sh
cd apps/web
npm run dev
```

Или вручную:

```bash
cd /Users/artur220513timur110415gmail.com/global-risk-platform/apps/web
rm -rf node_modules/.vite dist
npm run dev
```

## 3. Жёсткое обновление в браузере

Откройте http://127.0.0.1:5180/command и нажмите:

- **Mac:** Cmd+Shift+R  
- **Windows/Linux:** Ctrl+Shift+R  

(или DevTools → вкладка Network → включите "Disable cache" и обновите страницу).

## 4. Проверка, что подгрузился новый код

1. Включите чекбоксы **Flood**, **Heavy rain** и т.д. в панели SOURCE справа.
2. Откройте консоль браузера (F12 → Console).
3. Должно появиться сообщение вида:  
   `[CesiumGlobe] Climate markers: 15 cities at 5000m altitude (soft fill, no terrain occlusion)`  
   Если этого сообщения нет — всё ещё грузится старый бандл, повторите шаги 1–3.

## Что должно быть после применения

- **Все города по чекбоксам:** при включении Flood/Wind/Heat/Heavy rain/Drought/UV/Metro на глобусе отображаются все города этого слоя (расширенный список в `CesiumGlobe.tsx`).
- **Синие круги как риск-зоны:** мягкая заливка (alpha 0.2), без обводки, не «диски».
- **Круги не перекрываются рельефом:** красные и синие круги рисуются на высоте 5000 м над эллипсоидом, поэтому не обрезаются рельефом при зуме (Кабул, Сана, Пхеньян и т.д.).
