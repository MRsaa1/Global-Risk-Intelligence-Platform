# 🎨 Визуальная демонстрация Global Risk Platform

## 🌐 Система запущена!

### Доступ к системе

1. **Control Tower UI**: http://localhost:3000
2. **API Gateway**: http://localhost:8000
3. **Demo Page**: http://localhost:3000/demo

---

## 📊 Визуальные компоненты

### 1. Dashboard (Главная страница)
**URL**: http://localhost:3000

**Элементы**:
- 🎨 **Градиентные карточки метрик**:
  - Активные сценарии (фиолетовый градиент)
  - Расчеты в процессе (розовый градиент)
  - Завершенные расчеты (синий градиент)
  - Портфели (зеленый градиент)

- 📈 **Интерактивные графики**:
  - Line Chart: тренды Capital Ratio и LCR за 6 месяцев
  - Bar Chart: сравнение compliance по фреймворкам

- 📋 **Таблицы данных**:
  - Активные сценарии
  - Последние расчеты
  - Портфели с метриками

### 2. Demo Page (Демо-страница)
**URL**: http://localhost:3000/demo

**Особенности**:
- ✅ Не требует аутентификации
- ✅ Показывает все функции с демо-данными
- ✅ Графики трендов (30 дней)
- ✅ Карточки с ключевыми метриками
- ✅ Список сценариев, расчетов, портфелей

**Визуальные элементы**:
- 4 градиентные карточки с метриками
- Line Chart для трендов Capital Ratio и LCR
- Таблицы с данными
- Цветовая индикация статусов

### 3. Scenarios (Управление сценариями)
**URL**: http://localhost:3000/scenarios

**Функционал**:
- Таблица всех сценариев
- Кнопка "Create Scenario"
- Редактирование и удаление
- Статусы с цветовой индикацией

**Визуал**:
- Material-UI таблица
- Диалоги для создания/редактирования
- Chips для статусов

### 4. Calculations (Расчеты)
**URL**: http://localhost:3000/calculations

**Функционал**:
- Таблица всех расчетов
- Кнопка "Start Calculation"
- Real-time обновления статуса (WebSocket)
- Просмотр результатов
- Отмена расчетов

**Визуал**:
- Статусы с цветами (running=blue, completed=green, failed=red)
- Иконки действий
- Real-time обновления без перезагрузки

### 5. Calculation Detail (Детали расчета)
**URL**: http://localhost:3000/calculations/:id

**Вкладки**:
- 📈 **Overview**: 
  - Pie Chart: структура капитала
  - Bar Chart: метрики соответствия
  - Ключевые метрики в карточках

- 📊 **Basel IV**:
  - Таблица с результатами Basel IV
  - CET1 Ratio, Capital Requirement, RWA

- 💧 **Liquidity**:
  - Таблица с метриками ликвидности
  - LCR, HQLA, Net Cash Outflows

- 📋 **Details**:
  - Полный JSON результатов

**Кнопки экспорта**:
- Export PDF
- Export Excel

### 6. Portfolios (Портфели)
**URL**: http://localhost:3000/portfolios

**Визуал**:
- Таблица портфелей
- Агрегированные метрики
- Форматирование валют

## 🎯 Что можно протестировать прямо сейчас

1. **Откройте Demo Page**
   ```
   http://localhost:3000/demo
   ```
   - Увидите все функции без аутентификации
   - Графики и метрики
   - Демо-данные

2. **Войдите в систему**
   ```
   http://localhost:3000
   ```
   - Username: `demo`
   - Password: `demo`
   - Исследуйте все страницы

3. **Создайте сценарий**
   - Перейдите в Scenarios
   - Нажмите "Create Scenario"
   - Заполните форму

4. **Запустите расчет**
   - Перейдите в Calculations
   - Нажмите "Start Calculation"
   - Наблюдайте real-time обновления

5. **Просмотрите результаты**
   - После завершения расчета
   - Нажмите на иконку просмотра
   - Изучите графики и таблицы

6. **Экспортируйте отчет**
   - На странице результатов
   - Нажмите "Export PDF" или "Export Excel"
