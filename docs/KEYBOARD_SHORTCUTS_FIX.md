# Исправление горячих клавиш в Command Center

## ✅ Что было исправлено

**Проблема:** Горячие клавиши не работали, хотя подсказки отображались внизу экрана

**Причины:**
1. Обработчик был привязан только к `window`, а не к `document`
2. Слишком строгая проверка модификаторов блокировала все клавиши
3. Основной контейнер не мог получать фокус

**Решение:**
1. ✅ Обработчик привязан к `document` (catch все события)
2. ✅ Используется capture phase для приоритета
3. ✅ Правильная обработка модификаторов (блокирует только Cmd/Ctrl/Alt)
4. ✅ Добавлен `tabIndex={0}` на основной контейнер
5. ✅ Добавлено логирование для отладки

## 🔍 Как проверить, что работает

### Тест 1: Базовый тест

1. Откройте `http://localhost:5180/command`
2. Убедитесь, что фокус на странице (кликните в любом месте)
3. Нажмите клавишу **S**
4. В консоли должно появиться: `✅ Keyboard: S - Stress Test`
5. Должен запуститься stress test

### Тест 2: Проверка всех клавиш

1. Откройте консоль браузера (F12)
2. Нажмите каждую клавишу:
   - **S** → должно появиться `✅ Keyboard: S - Stress Test`
   - **D** → должно появиться `✅ Keyboard: D - Digital Twin`
   - **R** → должно появиться `✅ Keyboard: R - Reset View`
   - **Z** → должно появиться `✅ Keyboard: Z/N - Zone Navigation`
   - **ESC** → должно появиться `✅ Keyboard: ESC pressed - Back`

### Тест 3: Проверка блокировки в input

1. Если на странице есть input поле, кликните в него
2. Нажмите **S** → ничего не должно произойти (клавиша игнорируется)
3. Кликните вне input
4. Нажмите **S** → должен сработать shortcut

## 🐛 Отладка

Если клавиши все еще не работают:

1. **Откройте консоль браузера (F12)**
2. **Проверьте, что обработчик привязан:**
   - При загрузке страницы должно появиться: `✅ Keyboard shortcuts handler attached`

3. **Проверьте, что события доходят:**
   - Нажмите любую клавишу
   - В консоли должно появиться сообщение (если это обрабатываемая клавиша)

4. **Проверьте фокус:**
   - Убедитесь, что фокус не в input/textarea
   - Кликните в любом месте страницы
   - Попробуйте снова

5. **Проверьте конфликты:**
   - Убедитесь, что не нажаты Cmd/Ctrl/Alt
   - Попробуйте без модификаторов

## 📝 Технические детали

### Файл: `apps/web/src/pages/CommandCenter.tsx`

**Компонент:** `KeyboardHandler`

**Обработка:**
- События слушаются на `document` в capture phase
- Игнорируются, если фокус в input/textarea
- Блокируются Cmd/Ctrl/Alt комбинации (чтобы не конфликтовать с браузером)
- Разрешены простые клавиши и Shift+клавиша

**Приоритет:**
- Capture phase (`true`) = события обрабатываются до других обработчиков
- `preventDefault()` и `stopPropagation()` предотвращают конфликты
