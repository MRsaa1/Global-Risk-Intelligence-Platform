# Quantum Aesthetics Vision

## Инструмент мирового уровня с эстетикой квантовой физики

Этот документ задаёт направление: как сохранить идентичность **Decision Intelligence Platform** и одновременно поднять визуальный и концептуальный уровень до «мирового инструмента» с эстетикой, вдохновлённой квантовой физикой — без перехода в «научпоп» или декоративность.

---

## Принцип стыковки с IDENTITY.md

- **One Screen, Scene > Layout, Silence > Noise** остаются главными. Квантовая эстетика — не «красивые картинки», а **язык представления неопределённости, связности и наблюдателя**.
- Риск по природе вероятностен; «наблюдение» (фокус на регионе, стресс-тест) меняет контекст. Это уже квантовая метафора: наблюдатель и система связаны.
- Мировой уровень = **достоверность, сдержанность, точность**. Квантовая эстетика поддерживает это: глубокий фон, точечные «излучения», математическая чёткость типографики и сеток.

---

## 1. Визуальные метафоры из квантовой физики

| Концепт | Визуальная реализация | Где использовать |
|--------|------------------------|------------------|
| **Волновая функция / суперпозиция** | Не один цвет точки, а тонкий градиент или «облако» вероятности вокруг маркера риска | Hotspots на глобусе: ядро + размытая обводка по уровню uncertainty |
| **Коллапс при измерении** | При клике по региону — плавное «схлопывание» в один чёткий режим (панель контекста, один выбранный уровень) | Переход Global → Focused Analysis |
| **Запутанность** | Связи между регионами/активами не «линии», а синхронное пульсирование или общий тон при выделении одного узла | Графы зависимостей, cascade, risk network |
| **Неопределённость (Гейзенберг)** | Числа не «одно значение», а интервал или доверительная полоса, где уместно | Capital at Risk, Stress Loss P95, прогнозы |
| **Тёмное поле / вакуум** | Фон — не плоский чёрный, а глубокий градиент (тёмно-синий → чёрный) или едва заметная текстура «глубины» | Фон сцены, панелей, полноэкранных режимов |
| **Квант света / излучение** | Акценты — не плоские заливки, а лёгкое свечение (box-shadow, blur), редкие и смысловые | Критические алерты, выбранный hotspot, активный сценарий |

---

## 2. Цвет и свет

- **База**: сохраняем тёмную тему (zinc/black). Добавляем «глубину»:
  - Фон сцены: `#030712` → `#0a0a0f` (лёгкий синий оттенок), не чисто `#09090b`.
  - Панели: как сейчас, с опциональной очень тонкой границей «свечения» для активного состояния.
- **Акценты «квантового» луча** (редко, по действию):
  - Cyan/teal для «наблюдения», выбора: `#22d3ee` / `#06b6d4` с лёгким glow.
  - Фиолетовый для «расчёта», сценария: `#a78bfa` / `#8b5cf6`.
  - Красный/оранжевый — только для критики и алертов (как сейчас), но с возможностью мягкого glow вместо плоской заливки.
- **Правило**: один активный «квант» света на сцену (выбранный объект или один тип алерта). Остальное — приглушённо.

---

## 3. Типографика и данные

- **Моноширинный, научный тон**: JetBrains Mono уже задаёт «инструмент». Можно добавить второй шрифт только для крупных заголовков (например, **Instrument Sans** или **Geist**) — один вес, только для заголовков сцен.
- **Числа**: где есть uncertainty — показывать интервал или ±, например `€33.7M ± 2.1M` или `P95: €51.6M` с тултипом «90% CI: …».
- **Формулы/нотация**: не перегружать, но в документации и в подсказках можно использовать короткие обозначения (σ, ρ, P95, VaR) как часть языка платформы.

---

## 4. Движение и интеракция

- **Суперпозиция → коллапс**: при открытии панели контекста — короткая анимация «схлопывания» (opacity + scale), не «выезд сбоку».
- **Запутанность**: при hover/выборе узла в графе — связанные узлы слегка подсвечиваются или пульсируют с небольшой задержкой (stagger).
- **Частицы**: только если они несут смысл (например, поток данных, распространение риска). Цвет и плотность привязаны к данным; без декоративных «звёзд».
- **Тайминг**: быстрые переходы (150–250 ms) для смены контекста; чуть медленнее (300–400 ms) для появления/исчезновения панелей, чтобы не дёргать взгляд.

---

## 5. Звук (опционально)

- Мировой инструмент может иметь **очень тихие**, редкие звуки:
  - Подтверждение критического действия (один низкий тон).
  - Фоновый «пульс» только в режиме стресс-теста (минимально).
- Всё отключаемо по умолчанию; включается в настройках «Audio feedback».

---

## 6. Копирайт и нарратив

- В интерфейсе не писать «квантовый» ради бренда. Использовать точные термины:
  - «Вероятностная оценка», «Доверительный интервал», «Связанные риски», «Сценарий при заданном уровне».
- В одном предложении для маркетинга/лендинга можно сформулировать так:  
  *«Платформа решений в условиях неопределённости: один сценарный слой, вероятностные метрики, связь рисков в реальном времени.»*

## 7. Пошаговая реализация

Реализация выполнена по фазам 1–6 с привязкой к конкретным файлам (см. план в `.cursor/plans/`):

| Фаза | Действие | Результат |
|------|----------|-----------|
| **1. Токены** | Добавить в Tailwind/CSS переменные: `--quantum-bg-deep`, `--quantum-glow-cyan`, `--quantum-glow-violet`, `--quantum-uncertainty-stroke`. Заменить один-два фона сцены на градиент глубины. | Единая палитра «квантового» шага |
| **2. Глобус** | Hotspots: обводка с лёгким размытием (blur) и opacity по уровню риска; при выборе — один акцент с glow. | Суперпозиция → коллапс на сцене |
| **3. Панели** | Активная панель — тонкая граница glow (cyan/violet). Переход появления — лёгкий scale + opacity. | Чёткий «наблюдатель» и тихий интерфейс |
| **4. Метрики** | Где есть интервалы (P95, VaR, CaR) — тултип или подпись с доверительным интервалом; в таблицах — опциональная колонка ±. | Явная неопределённость |
| **5. Графы** | При выборе узла — подсвет связанных с небольшой задержкой; рёбра не перегружать. | Метафора запутанности |
| **6. Документация** | Обновить IDENTITY.md одним абзацем про «вероятностную эстетику и наблюдателя»; этот файл — расширение визуального языка. | Согласованность решений |

## 8. Чего избегать

- Яркие «космические» градиенты на весь экран.
- Много анимированных частиц без привязки к данным.
- Слово «квантовый» в UI.
- Перегруз анимациями (всё движется).
- Светлая тема как основная (контраст с «глубиной» и контрольными залами).

## Итог в одном предложении

**Инструмент мирового уровня с эстетикой квантовой физики = тёмная, глубокая сцена; риск как вероятность с явной неопределённостью; один акцент «света» на выбранный объект; типографика и метрики в духе точного прибора; движение только для смены контекста и связей — без декора.**

*Документ можно использовать как фильтр для дизайн-ревью и как основу для тикетов по дизайн-системе.*
