# Screenshots: GPU vs no-GPU

Папка для скриншотов, демонстрирующих разницу работы платформы **с GPU** и **без GPU**.

Готовых снимков в репозитории пока нет — их нужно снять по инструкции в [GPU_VS_NO_GPU_DOCUMENTATION.md](../GPU_VS_NO_GPU_DOCUMENTATION.md) (раздел 4).

Рекомендуемые имена файлов (после съёмки):

- `command-center-with-gpu.png` — Command Center при запущенном NIM на GPU (бейдж «NIM available» и т.п.).
- `command-center-without-gpu.png` — тот же экран без NIM / без GPU.
- `report-with-nim-block.png` — фрагмент отчёта стресс-теста с блоком погоды от NIM.
- `health-with-gpu.png` / `health-without-gpu.png` — ответ `/api/v1/health` при наличии/отсутствии NIM (опционально).

При добавлении файлов можно обновить ссылки в `GPU_VS_NO_GPU_DOCUMENTATION.md`.
