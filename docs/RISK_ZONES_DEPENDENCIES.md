# 🔗 Анализ зависимостей между критическими зонами риска

## Обзор

Анализ 15 критических зон риска (Risk: 83-100%) показывает **множественные прямые и косвенные зависимости** между ними. Эти зависимости создают **каскадные риски** и **скрытые экспозиции**.

---

## 📊 Карта зависимостей

### **КЛАСТЕР 1: Украинский конфликт** (Прямые военные связи)

#### **Прямые зависимости:**
1. **Kyiv** ↔ **Kharkiv** ↔ **Odesa** ↔ **Donetsk-Luhansk**
   - **Тип:** Военная операционная связь
   - **Критичность:** 0.95
   - **Механизм:** 
     - Общая линия фронта
     - Общая логистическая инфраструктура
     - Общие энергосети (Украинская энергосистема)
     - Общие транспортные коридоры
   - **Каскадный эффект:** Падение одной зоны → усиление давления на другие

2. **Donetsk-Luhansk** → **Kyiv/Kharkiv/Odesa**
   - **Тип:** Контроль территории → влияние на остальную Украину
   - **Критичность:** 0.90
   - **Механизм:** Контроль Донбасса влияет на:
     - Энергетическую безопасность всей Украины
     - Транспортные маршруты
     - Промышленное производство

#### **Косвенные зависимости:**
- **Энергетическая инфраструктура:** Все 4 зоны зависят от единой энергосистемы Украины
- **Транспорт:** Железнодорожные и автомобильные коридоры связывают все зоны
- **Финансовые потоки:** Общая валюта, банковская система

### **КЛАСТЕР 2: Ближневосточный конфликт** (Геополитические связи)

#### **Прямые зависимости:**
1. **Tel Aviv** ↔ **Gaza City** ↔ **Damascus** ↔ **Aleppo**
   - **Тип:** Военный конфликт + региональная стабильность
   - **Критичность:** 0.92
   - **Механизм:**
     - Израиль-Газа: Прямой военный конфликт
     - Сирия (Damascus, Aleppo): Региональная дестабилизация
     - Общие границы, беженцы, энергетические маршруты
   - **Каскадный эффект:** Эскалация в одной зоне → региональная нестабильность

2. **Damascus** ↔ **Aleppo**
   - **Тип:** Внутренний конфликт + контроль территории
   - **Критичность:** 0.95
   - **Механизм:** Обе зоны в Сирии, общая инфраструктура, общие военные операции

#### **Косвенные зависимости:**
- **Энергетические маршруты:** Трубопроводы через регион
- **Беженцы:** Миграционные потоки между зонами
- **Региональные альянсы:** Иран, Турция, Россия влияют на все зоны
- **Морские маршруты:** Средиземное море связывает Tel Aviv и сирийские порты

## 🎯 Критические пути каскадных эффектов

### **Путь 1: Украинский конфликт → Глобальная энергетика**
```
Donetsk-Luhansk (контроль) 
  → Энергосети Украины (нарушение)
    → Европейские энергорынки (дефицит)
      → Глобальные цены на энергоносители
        → Влияние на Caracas, все зоны
```

### **Путь 2: Ближневосточный конфликт → Региональная нестабильность**
```
Gaza City (эскалация)
  → Tel Aviv (ответ)
    → Региональная нестабильность
      → Damascus, Aleppo (эскалация)
        → Миграционные потоки
          → Sanaa, Khartoum, Tripoli (давление)
```

### **Путь 3: Тайваньский кризис → Глобальная торговля**
```
Taipei (кризис)
  → Тайваньский пролив (блокировка)
    → 50% мирового контейнерного трафика (нарушение)
      → Глобальные цепочки поставок
        → Влияние на все зоны (экономический кризис)
```

### **Путь 4: Африканские конфликты → Миграция → Региональная нестабильность**
```
Khartoum (конфликт)
  → Миграционные потоки
    → Tripoli (транзит)
      → Средиземное море (миграция)
        → Tel Aviv, Damascus (давление)
          → Эскалация региональных конфликтов
```
